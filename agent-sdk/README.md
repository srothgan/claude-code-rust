# claude-rs agent-sdk bridge

Initial scaffold for the NDJSON stdio bridge that will connect Rust (`claude-code-rust`) with `@anthropic-ai/claude-agent-sdk`.

## Local build

```bash
npm install
npm run build
```

Build output is written to `dist/bridge.mjs`.
